Download & Installation

The latest release is {{page.release}}. It should be considered alpha quality.

Ubuntu & Debian

Install qdb-server_{{ page.release }}.deb. This includes init scripts, creates a qdb user and so on.

{% highlight bash %} $ wget http://downloads.qdb.io/qdb-server_{{ page.release }}.deb $ sudo apt-get install openjdk-7-jdk $ sudo dpkg -i qdb-server_{{ page.release }}.deb $ tail -f /var/log/qdb/qdb.log {% endhighlight %}

Generic

Install Java 6 or 7, unpack qdb-server-{{ page.release }}.tgz somewhere and run start.sh (to start QDB in the background) or run.sh to run it in the foreground.
